# NL2TDL Converter - Example Natural Language Commands
# 이 파일의 명령어들을 배치 모드로 테스트할 수 있습니다.
# 각 줄은 하나의 자연어 명령으로 처리됩니다.
# '#'로 시작하는 줄은 주석으로 무시됩니다.

# 기본 동작 예제
로봇을 홈 위치로 이동시켜
로봇을 관절 각도 (0, 0, 90, 0, 90, 0)으로 이동
위치 (300, 200, 150)으로 직선 이동해줘

# I/O 제어 예제
디지털 출력 포트 1번을 켜줘
디지털 출력 포트 1번을 꺼줘
디지털 입력 포트 11번이 켜질 때까지 10초 동안 대기

# 시간 지연 예제
2초 대기
5초간 정지

# 복합 작업 예제
로봇 속도를 50%로 설정하고 위치 (400, 300, 200)으로 이동
디지털 출력 1번을 켜고 3초 대기한 후 다시 꺼줘
홈 위치로 이동한 후 2초 대기하고 종료

# 로봇 설정 예제
관절 속도를 60%로 설정
작업 속도를 100mm/s로 설정
툴 번호 1번으로 변경

# 용접 작업 예제
아크 용접을 시작해줘
아크 용접을 종료해줘

# 제어 흐름 예제 (간단한 경우)
1초 대기하고 출력 포트 2번을 켜줘

# 순차 작업 예제
포트 1을 켜고 위치 A로 이동한 후 포트 1을 꺼줘
